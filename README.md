# テトリス

HTMLで動作するクラシックなテトリスゲームです。

## 機能

- 🎮 7種類のテトロミノ（I, O, T, S, Z, J, L）
- 📊 スコア計算とレベルシステム
- 🎯 ネクストピース表示
- ⏸️ ポーズ機能
- 🎨 カラフルなデザイン
- 📱 レスポンシブデザイン

## 操作方法

| キー | 操作 |
|------|------|
| ← → | ブロックを左右に移動 |
| ↑ | ブロックを回転 |
| ↓ | ブロックを高速落下 |
| Space | 一時停止・再開 |

## ボタン

- **スタート**: ゲーム開始
- **ポーズ**: 一時停止・再開
- **リセット**: ゲームリセット

## ゲームルール

1. ブロックが上から落下してきます
2. 左右キーでブロックを移動、上キーで回転させます
3. 横1行完全に埋まるとその行が消えます
4. ブロックが画面上部に達したらゲームオーバーです

## スコア計算

- 1行消去: 100点
- 2行消去: 400点
- 3行消去: 900点
- 4行消去: 1600点

10行消去するごとにレベルが上がり、ブロックが早くなります。

## ファイル構成

- `index.html` - HTMLページ
- `style.css` - スタイル
- `tetris.js` - ゲームロジック

## 使い方

1. `index.html`をブラウザで開く
2. 「スタート」ボタンをクリック
3. キーボードでゲームをプレイ

楽しいテトリスゲームをお楽しみください！🎮
